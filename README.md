# Grid Designer (Vanilla JS Starter)

Simple vanilla JavaScript grid designer for quickly prototyping CSS grid layouts.

## Quick start

1. Open `index.html` in your browser.
2. Set columns/rows and click `Build Grid`.
3. Click cells to toggle active state.
4. Click `Export CSS` to download a CSS file for your layout.

## Project files

- `index.html` – UI skeleton
- `style.css` – Base layout and cell styles
- `script.js` – Grid generation, interaction, export
- `README.md` – Project instructions
